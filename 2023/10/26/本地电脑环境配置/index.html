本地电脑环境配置

引言

本文是为了记录本人的笔记本电脑环境配置，以便于以后换电脑或者重装系统时能够快速配置环境。包含软件的安装、环境配置、软件配置等内容。同时也作为推荐。

软件安装

Scoop

一个Windows上的命令行包管理器,类似于Linux上的apt-get或者Windows上的winget。它可以用来安装和管理软件。
需运行在Windows powershell 5.1以上版本，可通过$PSVersionTable.PSVersion查看powershell版本。
安装命令如下（更多安装方式见官方 )：

推荐在自定义的目录下安装，可防止中文用户名导致的软件执行问题

  1. 常用命令（全部命令）：
    • bucket相关命令
      • scoop bucket add <bucket_name> <bucket_url>：添加别名为bucket_name的bucket源
      • scoop bucket rm <bucket_name>：删除bucket源
      • scoop update：更新bucket源
    • 软件安装相关命令 (软件包下载后会安装到本地的Scoop/apps目录下)
      • scoop install <app_name>：安装软件
      • scoop uninstall <app_name>：卸载软件
      • scoop update <app_name>：更新软件
      • scoop cleanup <app_name>：清理旧版本软件
      • scoop reset <app_name>@<version>：回滚软件到指定版本
      • scoop info <app_name>：查看软件简要信息
      • scoop hold <app_name>：锁定软件版本，防止更新
      • scoop list：列出已安装软件
      • scoop search <app_name>：搜索软件 (不推荐这个命令，速度很慢，推荐使用下面推荐的scoop-search / scoop-search-multisource,或到官网搜索)
    • cache相关命令 (软件包下载后会缓存到本地的Scoop/cache目录下)
      • scoop cache rm <app_name>：清理软件缓存
      • scoop cache rm * ：清理所有软件缓存
      • scoop cache：列出下载的软件包
  2. Scoop目录下子目录含义：
    • apps——所有通过scoop安装的软件都在里面。
    • buckets——管理软件的仓库，用于记录哪些软件可以安装、更新等信息，默认添加main仓库，主要包含无需GUI的软件。
    • cache——软件下载后安装包暂存目录。
    • persit——用于储存一些用户数据，不会随软件更新而替换。
    • shims——用于软链接应用，使应用之间不会互相干扰，实际使用过程中无用户操作不必细究
  3. bucket源推荐：
    • 官方镜像源(及南大源)及官方介绍：
      • main - Default bucket for the most common (mostly CLI) apps
      • extras - Apps that don’t fit the main bucket’s criteria
      • nerd-fonts - Nerd Fonts
      • nirsoft - Almost all of the 250+ apps from Nirsoft
      • java - A collection of Java development kits (JDKs), Java runtime engines (JREs), Java’s virtual machine debugging tools and Java based runtime engines.
      • nonportable - Non-portable apps (may require UAC)
      • versions - Alternative versions of apps found in other buckets
    • 其他软件源：
      • okibcn/ScoopMaster:包含4万多个软件，基本上能满足所有需求，但有时下载的软件过不了MD5校验，需要手动下载安装包；

        Scoop Master, the way to keep all the apps in Scoop ultra-updated. One bucket for all.

      • lzwme/scoop-proxy-cn:包含9千多个软件，

        适合中国大陆用户使用的 Scoop buckets 代理镜像库。从多个开源 bucket 仓库同步更新，并将从 github release 下载的应用地址修改为基于 ghporxy.com 的代理下载地址。

      • kkzzhizhou/scoop-apps:包含1万多个软件，合并了多个bucket源
  4. 推荐软件(均可从scoop下载)
    • git：bucket基本上都需要git，所以必须安装
    • 7zip：下载软件压缩包的必需解压缩软件。
    • aria2：多线程下载器，下载软件的必需下载软件,可提升下载速度。
    • scoop-search：scoop search命令的替代品，速度更快，且支持模糊搜索,推荐设置别名ss: New-Alias ss scoop-search。
    • scoop-search-multisource：scoop search命令的替代品，速度更快，且支持模糊搜索,可搜索网络上的多个bucket源，推荐设置别名ssm: New-Alias ssm scoop-search-multisource。
    • fd: find的替代品，速度更快，且支持模糊搜索。
    • fiddler：网络抓包工具，可使Windows UWP应用直接走本机DNS服务器，解决UWP应用(如onenote,onedrive等)连接微软DNS服务器导致无法连接的问题。
    • sudo: 可在命令前加上sudo，临时将命令权限提升至管理员;
